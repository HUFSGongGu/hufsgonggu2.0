# Setup

```sh
# 0. clone project
git clone https://github.com/HUFSGongGu/hufsgonggu2.0.git

# 1. virtual environment setup
python -m venv myvenv

# 2. venv run
# windows
source myvenv/Scripts/activate

# MacOS
source myvenv/bin/activate

# 3. runserver
cd hgg
python manage.py runserver
```
